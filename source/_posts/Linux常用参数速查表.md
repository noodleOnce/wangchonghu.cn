## 📁 一、目录/文件操作

### 1️⃣ mkdir

``` bash
mkdir -p dir
```

-   `-p`：递归创建目录（最常用）

------------------------------------------------------------------------

### 2️⃣ cp

``` bash
cp -r src dest
cp -f file1 file2
cp -v file1 file2
```

-   `-r`：递归复制目录\
-   `-f`：强制覆盖\
-   `-v`：显示复制过程（verbose）

------------------------------------------------------------------------

### 3️⃣ mv

``` bash
mv -f a.txt b.txt
mv -v a.txt b.txt
```

-   `-f`：强制覆盖\
-   `-v`：显示移动过程

------------------------------------------------------------------------

### 4️⃣ rm

``` bash
rm -rf dir
rm -f file
```

-   `-r`：递归删除\
-   `-f`：强制删除（不提示）

⚠️ `rm -rf`：高危命令（慎用）

------------------------------------------------------------------------

## 🔍 二、查看/查找

### 5️⃣ ls

``` bash
ls -l
ls -a
ls -lh
```

-   `-l`：详细信息\
-   `-a`：显示隐藏文件\
-   `-h`：人类可读（KB/MB）

------------------------------------------------------------------------

### 6️⃣ grep

``` bash
grep -r "xxx" .
grep -n "xxx" file
grep -i "xxx" file
```

-   `-r`：递归搜索\
-   `-n`：显示行号\
-   `-i`：忽略大小写

------------------------------------------------------------------------

### 7️⃣ find

``` bash
find . -name "*.log"
find . -type d
```

-   `-name`：按名称查找\
-   `-type d`：目录\
-   `-type f`：文件

------------------------------------------------------------------------

## 📄 三、文件内容

### 8️⃣ cat

``` bash
cat file.txt
```

👉 查看整个文件（适合小文件）

------------------------------------------------------------------------

### 9️⃣ less

``` bash
less file.txt
```

👉 分页查看（推荐）

------------------------------------------------------------------------

### 🔟 tail

``` bash
tail -f app.log
tail -n 100 app.log
```

-   `-f`：实时查看日志（开发必备）\
-   `-n`：查看最后 N 行

------------------------------------------------------------------------

## 🌐 四、网络相关

### 1️⃣1️⃣ curl

``` bash
curl -X POST url
curl -H "Content-Type: application/json" url
```

-   `-X`：指定请求方法\
-   `-H`：设置请求头\
-   `-d`：发送数据

------------------------------------------------------------------------

### 1️⃣2️⃣ wget

``` bash
wget url
wget -O file url
```

-   `-O`：指定输出文件名

------------------------------------------------------------------------

## 🔐 五、权限相关

### 1️⃣3️⃣ chmod

``` bash
chmod -R 755 dir
```

-   `-R`：递归修改权限

------------------------------------------------------------------------

### 1️⃣4️⃣ chown

``` bash
chown -R user:user dir
```

-   `-R`：递归修改所属用户

------------------------------------------------------------------------

## ⚙️ 六、压缩解压

### 1️⃣5️⃣ tar

``` bash
tar -czf file.tar.gz dir
tar -xzf file.tar.gz
```

-   `-c`：创建\
-   `-x`：解压\
-   `-z`：gzip\
-   `-f`：指定文件

👉 常见组合： - `-czf`：打包压缩\
- `-xzf`：解压

------------------------------------------------------------------------

## 🧠 高频参数总结（重点记忆）

  参数   含义             高频命令
  ------ ---------------- ----------------
  `-p`   递归创建目录     mkdir
  `-r`   递归操作         cp / rm / grep
  `-f`   强制执行         rm / cp / mv
  `-v`   显示过程         cp / mv
  `-n`   行号/数量        grep / tail
  `-h`   人类可读         ls
  `-R`   递归（权限类）   chmod / chown

------------------------------------------------------------------------

## 💡 实战建议

### ✅ 1. 必会三板斧

``` bash
tail -f 日志
grep 关键字
find 查文件
```

------------------------------------------------------------------------

### ✅ 2. 日志排查黄金组合

``` bash
grep "ERROR" app.log | tail -n 50
```

------------------------------------------------------------------------

### ✅ 3. 安全删除习惯

``` bash
rm -rf 前先 ls 看一眼
```

------------------------------------------------------------------------

### ✅ 4. 写脚本一定用 `-p`

``` bash
mkdir -p /xxx/xxx
```

👉 保证脚本可重复执行（幂等）

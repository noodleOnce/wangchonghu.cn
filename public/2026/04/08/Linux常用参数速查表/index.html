Linux常用参数速查表

2026-04-08 工具 评论

📁 一、目录/文件操作

1️⃣ mkdir

  • -p：递归创建目录（最常用）

2️⃣ cp

  • -r：递归复制目录\
  • -f：强制覆盖\
  • -v：显示复制过程（verbose）

3️⃣ mv

  • -f：强制覆盖\
  • -v：显示移动过程

4️⃣ rm

  • -r：递归删除\
  • -f：强制删除（不提示）

⚠️ rm -rf：高危命令（慎用）

🔍 二、查看/查找

5️⃣ ls

  • -l：详细信息\
  • -a：显示隐藏文件\
  • -h：人类可读（KB/MB）

6️⃣ grep

  • -r：递归搜索\
  • -n：显示行号\
  • -i：忽略大小写

7️⃣ find

  • -name：按名称查找\
  • -type d：目录\
  • -type f：文件

📄 三、文件内容

8️⃣ cat

👉 查看整个文件（适合小文件）

9️⃣ less

👉 分页查看（推荐）

🔟 tail

  • -f：实时查看日志（开发必备）\
  • -n：查看最后 N 行

🌐 四、网络相关

1️⃣1️⃣ curl

  • -X：指定请求方法\
  • -H：设置请求头\
  • -d：发送数据

1️⃣2️⃣ wget

  • -O：指定输出文件名

🔐 五、权限相关

1️⃣3️⃣ chmod

  • -R：递归修改权限

1️⃣4️⃣ chown

  • -R：递归修改所属用户

⚙️ 六、压缩解压

1️⃣5️⃣ tar

  • -c：创建\
  • -x：解压\
  • -z：gzip\
  • -f：指定文件

👉 常见组合： - -czf：打包压缩\

  • -xzf：解压

🧠 高频参数总结（重点记忆）

参数 含义 高频命令

-p 递归创建目录 mkdir
-r 递归操作 cp / rm / grep
-f 强制执行 rm / cp / mv
-v 显示过程 cp / mv
-n 行号/数量 grep / tail
-h 人类可读 ls
-R 递归（权限类） chmod / chown

💡 实战建议

✅ 1. 必会三板斧

✅ 2. 日志排查黄金组合

✅ 3. 安全删除习惯

✅ 4. 写脚本一定用 -p

👉 保证脚本可重复执行（幂等）
